# VoiceSSH

Android εφαρμογή για έλεγχο απομακρυσμένων Windows συστημάτων μέσω φωνητικών εντολών και SSH.

## 📋 Περιγραφή

Το **VoiceSSH** είναι μια εφαρμογή Android που επιτρέπει τον απομακρυσμένο έλεγχο Windows συστημάτων μέσω φωνητικών εντολών. Η εφαρμογή μετατρέπει φωνητικές εντολές (Ελληνικά) σε εντολές που εκτελούνται μέσω SSH στους επιλεγμένους servers και παρέχει φωνητική ανατροφοδότηση (TTS) για επιβεβαίωση ή αναφορά αποτελεσμάτων.

Η κύρια ιδέα του project είναι να προσφέρει έναν γρήγορο και φιλικό τρόπο ελέγχου μηχανημάτων Windows από κινητές συσκευές Android, σε περιβάλλοντα όπου ο χρήστης χρειάζεται άμεση εκτέλεση εντολών χωρίς φυσική πρόσβαση στον υπολογιστή.

### ✨ Χαρακτηριστικά

- 🎤 Φωνητική Αναγνώριση: Εκτέλεση εντολών με φωνητικές εντολές (Ελληνικά).
- 🔊 Φωνητική Ανατροφοδότηση: Text-to-Speech (TTS) για επιβεβαίωση εντολών και αποτελεσμάτων.
- 🖥️ Πολλαπλοί SSH Servers: Υποστήριξη πολλαπλών SSH συνδέσεων και εκτέλεση εντολών σε πολλούς servers ταυτόχρονα.
- 💾 Διαχείριση Βάσης Δεδομένων: Import/Export ρυθμίσεων και εντολών σε JSON.
- 📱 Material Design UI: Σύγχρονη διεπαφή με KivyMD.
- 🔐 Ασφαλής Σύνδεση: Υποστήριξη password και key authentication για SSH.

## 🚀 Εγκατάσταση

### Προαπαιτούμενα

#### Για την εφαρμογή Android:
- Android συσκευή με API 21+ (Android 5.0 Lollipop ή νεότερο)
- Δικαιώματα για μικρόφωνο και αποθηκευτικό χώρο

#### Για τα απομακρυσμένα Windows συστήματα:
- SSH Server (π.χ. OpenSSH για Windows)
- Ενεργοποιημένες υπηρεσίες SSH
- Έγκυρα credentials (username/password ή SSH key)

### Εγκατάσταση APK

1. Κατεβάστε το αρχείο APK από τον φάκελο `bin/` (αν υπάρχει)
2. Ενεργοποιήστε την εγκατάσταση από άγνωστες πηγές στη συσκευή σας
3. Εγκαταστήστε το APK
4. Δώστε τα απαραίτητα δικαιώματα (μικρόφωνο, αποθηκευτικός χώρος)

### Build από τον κώδικα

```bash

git clone https://github.com/spyalekos/androidis.git
cd androidis

# Εγκατάσταση dependencies (παράδειγμα)
pip install -r requirements.txt

# Build APK με buildozer
buildozer android debug
```

## 📖 Χρήση

### Πρώτη Εκκίνηση

1. Ρύθμιση SSH Σύνδεσης:
   - Πατήστε το κουμπί μενού (☰) και επιλέξτε "Ρυθμίσεις"
   - Προσθέστε μια νέα SSH σύνδεση με: Alias, IP/hostname, Port (22), Username, Password ή SSH key

2. Δημιουργία Εντολών:
   - Από το κεντρικό μενού, επιλέξτε τη λίστα εντολών
   - Προσθέστε νέες εντολές με: Φωνητική εντολή (π.χ. "άνοιξε σημειωματάριο"), Εκτελέσιμη εντολή (π.χ. `notepad.exe`), Επιλογή SSH server(s)

### Εκτέλεση Φωνητικών Εντολών

1. Πατήστε το κουμπί μικροφώνου (🎤)
2. Πείτε την φωνητική εντολή
3. Η εφαρμογή θα αναγνωρίσει την εντολή και θα την εκτελέσει στους επιλεγμένους servers, και θα σας ενημερώσει φωνητικά για το αποτέλεσμα

### Παραδείγματα Εντολών

| Φωνητική Εντολή | Εκτελέσιμη Εντολή | Περιγραφή |
|----------------|-------------------|-----------|
| "άνοιξε σημειωματάριο" | `notepad.exe` | Άνοιγμα Notepad |
| "άνοιξε αριθμομηχανή" | `calc.exe` | Άνοιγμα Αριθμομηχανής |
| "εκτέλεση script" | `C:\\Scripts\\backup.bat` | Εκτέλεση batch script |
| "έλεγχος δικτύου" | `ipconfig /all` | Εμφάνιση πληροφοριών δικτύου |

## ⚙️ Ρυθμίσεις

### Διαχείριση SSH Συνδέσεων

- Προσθήκη: Κουμπί "+" για νέα σύνδεση
- Επεξεργασία: Πατήστε πάνω σε μια σύνδεση για αλλαγές
- Διαγραφή: Swipe ή κουμπί διαγραφής
- Δοκιμή: Δοκιμάστε τη σύνδεση πριν την αποθήκευση

### Διαχείριση Εντολών

- Προσθήκη: Δημιουργία νέων φωνητικών εντολών
- Επεξεργασία: Τροποποίηση υπαρχουσών εντολών
- Διαγραφή: Αφαίρεση εντολών
- Πολλαπλοί Servers: Επιλογή σε ποιους servers θα εκτελεστεί η εντολή

### Import/Export Δεδομένων

#### Export:
1. Μενού → Ρυθμίσεις → Export
2. Επιλέξτε τοποθεσία αποθήκευσης
3. Αποθήκευση JSON αρχείου με όλες τις ρυθμίσεις

#### Import:
1. Μενού → Ρυθμίσεις → Import
2. Επιλέξτε αρχείο JSON
3. Επιλέξτε λειτουργία: Αντικατάσταση ή Συγχώνευση

## 🛠️ Τεχνικές Λεπτομέρειες

### Τεχνολογίες

- Framework: Kivy + KivyMD
- Γλώσσα: Python 3
- SSH: Paramiko
- Database: SQLite3
- Android APIs: SpeechRecognizer, TextToSpeech, ContentResolver

### Δομή Βάσης Δεδομένων

Πίνακας `commands`:
- `id`: Μοναδικό αναγνωριστικό
- `name`: Φωνητική εντολή
- `executable`: Εκτελέσιμη εντολή

Πίνακας `ssh_connections`:
- `id`: Μοναδικό αναγνωριστικό
- `alias`: Όνομα σύνδεσης
- `host`: IP/hostname
- `port`: SSH port
- `username`: Όνομα χρήστη
- `password`: Κωδικός πρόσβασης (προτείνεται χρήση key αντί για plain password)

Πίνακας `command_ssh_mappings`:
- `command_id`: Αναφορά σε εντολή
- `ssh_alias`: Αναφορά σε SSH σύνδεση

## 🐛 Αντιμετώπιση Προβλημάτων

### Η εφαρμογή δεν αναγνωρίζει φωνή
- Βεβαιωθείτε ότι έχετε δώσει δικαιώματα μικροφώνου
- Έλεγχος ότι η γλώσσα συσκευής/φωνητικής αναγνώρισης είναι Ελληνικά
- Δοκιμάστε να μιλήσετε καθαρά και δυνατά

### SSH σύνδεση αποτυγχάνει
- Έλεγχος IP/hostname και port
- Βεβαιωθείτε ότι ο SSH server είναι ενεργός
- Έλεγχος username/password ή keys
- Έλεγχος firewall ρυθμίσεων
