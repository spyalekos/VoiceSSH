# VoiceSSH

Εφαρμογή Android για έλεγχο απομακρυσμένων Windows συστημάτων μέσω φωνητικών εντολών και SSH.

## 📋 Περιγραφή

Το **VoiceSSH** είναι μια εφαρμογή Android που σας επιτρέπει να ελέγχετε απομακρυσμένα Windows συστήματα χρησιμοποιώντας φωνητικές εντολές. Η εφαρμογή χρησιμοποιεί αναγνώριση φωνής για να λάβει εντολές, τις εκτελεί μέσω SSH σε έναν ή περισσότερους servers, και παρέχει φωνητική ανατροφοδότηση για το αποτέλεσμα.

### ✨ Χαρακτηριστικά

- 🎤 **Φωνητική Αναγνώριση**: Εκτέλεση εντολών με φωνητικές εντολές στα Ελληνικά
- 🔊 **Φωνητική Ανατροφοδότηση**: Text-to-Speech (TTS) για επιβεβαίωση εντολών και αποτελεσμάτων
- 🖥️ **Πολλαπλοί SSH Servers**: Υποστήριξη πολλαπλών SSH συνδέσεων και εκτέλεση εντολών σε πολλούς servers
- 💾 **Διαχείριση Βάσης Δεδομένων**: Import/Export ρυθμίσεων και εντολών
- 📱 **Material Design UI**: Σύγχρονη διεπαφή με KivyMD
- 🔐 **Ασφαλής Σύνδεση**: Χρήση SSH με υποστήριξη password και key authentication

## 🚀 Εγκατάσταση

### Προαπαιτούμενα

#### Για την εφαρμογή Android:
- Android συσκευή με API 21+ (Android 5.0 Lollipop ή νεότερο)
- Δικαιώματα για μικρόφωνο και αποθηκευτικό χώρο

#### Για τα απομακρυσμένα Windows συστήματα:
- SSH Server (π.χ. OpenSSH για Windows)
- Ενεργοποιημένες υπηρεσίες SSH
- Έγκυρα credentials (username/password ή SSH key)

### Εγκατάσταση APK

1. Κατεβάστε το αρχείο APK από τον φάκελο `bin/`
2. Ενεργοποιήστε την εγκατάσταση από άγνωστες πηγές στη συσκευή σας
3. Εγκαταστήστε το APK
4. Δώστε τα απαραίτητα δικαιώματα (μικρόφωνο, αποθηκευτικός χώρος)

### Build από τον κώδικα


## 📖 Χρήση

### Πρώτη Εκκίνηση

1. **Ρύθμιση SSH Σύνδεσης**:
   - Πατήστε το κουμπί μενού (☰) και επιλέξτε "Ρυθμίσεις"
   - Προσθέστε μια νέα SSH σύνδεση με:
     - Alias (όνομα αναγνώρισης)
     - IP διεύθυνση ή hostname
     - Port (συνήθως 22)
     - Username
     - Password

2. **Δημιουργία Εντολών**:
   - Από το κεντρικό μενού, επιλέξτε "Λίστα Προσταγμάτων"
   - Προσθέστε νέες εντολές με:
     - Φωνητική εντολή (π.χ. "άνοιξε σημειωματάριο")
     - Εντολή προς εκτέλεση (π.χ. "notepad.exe")
     - Επιλογή SSH server(s) για εκτέλεση

### Εκτέλεση Φωνητικών Εντολών

1. Πατήστε το κουμπί μικροφώνου (🎤) στην κεντρική οθόνη
2. Ακούστε το "πείτε μου" από την εφαρμογή
3. Πείτε τη φωνητική εντολή
4. Η εφαρμογή θα:
   - Αναγνωρίσει την εντολή
   - Την εκτελέσει στους επιλεγμένους servers
   - Σας ενημερώσει φωνητικά για το αποτέλεσμα

### Παραδείγματα Εντολών

| Φωνητική Εντολή | Εκτελέσιμη Εντολή | Περιγραφή |
|----------------|-------------------|-----------|
| "άνοιξε σημειωματάριο" | `notepad.exe` | Άνοιγμα Notepad |
| "άνοιξε υπολογιστή" | `calc.exe` | Άνοιγμα Αριθμομηχανής |
| "εκτέλεση script" | `C:\Scripts\backup.bat` | Εκτέλεση batch script |
| "έλεγχος δικτύου" | `ipconfig /all` | Εμφάνιση πληροφοριών δικτύου |

### Φωνητική Ανατροφοδότηση

Η εφαρμογή παρέχει φωνητική ανατροφοδότηση:
- **"πείτε μου"**: Όταν ενεργοποιείται η αναγνώριση φωνής
- **"η εντολή [όνομα] εκτελέστηκε επιτυχώς"**: Όταν η εντολή ολοκληρώνεται επιτυχώς
- **"υπάρχει πρόβλημα"**: Όταν υπάρχει σφάλμα εκτέλεσης

## ⚙️ Ρυθμίσεις

### Διαχείριση SSH Συνδέσεων

- **Προσθήκη**: Κουμπί "+" για νέα σύνδεση
- **Επεξεργασία**: Πατήστε πάνω σε μια σύνδεση για αλλαγές
- **Διαγραφή**: Swipe ή κουμπί διαγραφής
- **Δοκιμή**: Δοκιμάστε τη σύνδεση πριν την αποθήκευση

### Διαχείριση Εντολών

- **Προσθήκη**: Δημιουργία νέων φωνητικών εντολών
- **Επεξεργασία**: Τροποποίηση υπαρχουσών εντολών
- **Διαγραφή**: Αφαίρεση εντολών
- **Πολλαπλοί Servers**: Επιλογή σε ποιους servers θα εκτελεστεί η εντολή

### Import/Export Δεδομένων

#### Export:
1. Μενού → Ρυθμίσεις → Export
2. Επιλέξτε τοποθεσία αποθήκευσης
3. Αποθήκευση JSON αρχείου με όλες τις ρυθμίσεις

#### Import:
1. Μενού → Ρυθμίσεις → Import
2. Επιλέξτε αρχείο JSON
3. Επιλέξτε λειτουργία:
   - **Αντικατάσταση**: Διαγραφή όλων και φόρτωση νέων
   - **Συγχώνευση**: Προσθήκη στα υπάρχοντα

## 🛠️ Τεχνικές Λεπτομέρειες

### Τεχνολογίες

- **Framework**: Kivy + KivyMD
- **Γλώσσα**: Python 3
- **SSH**: Paramiko
- **Database**: SQLite3
- **Android APIs**: 
  - SpeechRecognizer για φωνητική αναγνώριση
  - TextToSpeech για φωνητική ανατροφοδότηση
  - ContentResolver για διαχείριση αρχείων

### Δομή Βάσης Δεδομένων

#### Πίνακας `commands`:
- `id`: Μοναδικό αναγνωριστικό
- `name`: Φωνητική εντολή
- `executable`: Εκτελέσιμη εντολή

#### Πίνακας `ssh_connections`:
- `id`: Μοναδικό αναγνωριστικό
- `alias`: Όνομα σύνδεσης
- `host`: IP/hostname
- `port`: SSH port
- `username`: Όνομα χρήστη
- `password`: Κωδικός πρόσβασης

#### Πίνακας `command_ssh_mappings`:
- `command_id`: Αναφορά σε εντολή
- `ssh_alias`: Αναφορά σε SSH σύνδεση

### Δικαιώματα Android

```xml
RECORD_AUDIO          - Για φωνητική αναγνώριση
INTERNET              - Για SSH συνδέσεις
READ_EXTERNAL_STORAGE - Για import αρχείων
WRITE_EXTERNAL_STORAGE - Για export αρχείων
```

## 🐛 Αντιμετώπιση Προβλημάτων

### Η εφαρμογή δεν αναγνωρίζει φωνή
- Βεβαιωθείτε ότι έχετε δώσει δικαιώματα μικροφώνου
- Έλεγχος ότι η γλώσσα συσκευής είναι Ελληνικά
- Δοκιμάστε να μιλήσετε καθαρά και δυνατά

### SSH σύνδεση αποτυγχάνει
- Έλεγχος IP/hostname και port
- Βεβαιωθείτε ότι ο SSH server είναι ενεργός
- Έλεγχος username/password
- Έλεγχος firewall ρυθμίσεων

### Import αρχείων δεν λειτουργεί
- Βεβαιωθείτε ότι έχετε δώσει δικαιώματα αποθήκευσης
- Χρησιμοποιήστε τον file manager για επιλογή αρχείων
- Έλεγχος ότι το JSON αρχείο είναι έγκυρο

### TTS δεν μιλάει
- Έλεγχος ότι η Google TTS είναι εγκατεστημένη
- Βεβαιωθείτε ότι η ελληνική γλώσσα είναι εγκατεστημένη για TTS
- Έλεγχος ότι η ένταση δεν είναι στο μηδέν
